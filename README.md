# Sistema de Registro de Acessórios PRO 🚀

Sistema moderno para controle de saída e retorno de acessórios de venda, desenvolvido com **React** e **Vite**.

## ✨ Funcionalidades

- **Registro de Movimentação**: Controle de saídas e retornos com Código de Fábrica, Nome Comercial, Responsável e O.S.
- **Catálogo de Itens**: Cadastro e gerenciamento de acessórios e responsáveis.
- **Dashboard de Métricas**: Visualização gráfica do desempenho por produto e por membro da equipe.
- **Exportação Excel**: Download do histórico completo em formato `.xlsx`.
- **Tema Escuro**: Interface moderna, limpa e profissional.
- **Persistência Local**: Dados salvos automaticamente no navegador (LocalStorage).

## 🛠️ Tecnologias Utilizadas

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Lucide React](https://lucide.dev/) (Ícones)
- [Recharts](https://recharts.org/) (Gráficos)
- [SheetJS](https://sheetjs.com/) (Excel)

## 🚀 Como rodar o projeto

1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
